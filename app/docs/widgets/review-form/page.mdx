# Review Form

Collects new customer reviews directly on product pages.

## When to use

Use this widget when you want on-page review submission without redirecting users.

## What you can customize

- Form labels and placeholder copy
- Success and error messaging
- Form structure and visual style

## Publish behavior

New submissions follow your app publishing preferences and moderation flow.

If you prefer manual moderation, keep auto-publish disabled.

You can manage this behavior in <a href={`${process.env.NEXT_PUBLIC_APP_ADMIN_URL}/settings`} target="_blank" rel="noopener">app settings</a>.

## Troubleshooting

- If submit fails, test with minimal required fields first.
- If reviews do not appear after submit, check moderation/auto-publish settings.
- If duplicate submits happen, verify the theme does not mount the widget twice.
