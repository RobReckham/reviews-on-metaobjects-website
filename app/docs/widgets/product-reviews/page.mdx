# Product Reviews

Displays a detailed list of reviews on product pages.

## When to use

Use this as your main social proof area where visitors can read real customer feedback.

## How it renders

- Shows reviews associated with the product.
- Prioritizes readable, chronological review browsing.
- Supports flexible layout and styling controls.

## What you can customize

- Number of visible reviews
- Section heading and empty state text
- Review card layout and visual style
- Show/hide behavior when no reviews are available

## Notes

- Keep this section close to product details to reduce friction.
- Pair with Product Rating for summary + deep dive flow.

## Troubleshooting

- If the list is empty, verify reviews are synced/imported and linked to products.
- If outdated data appears, run a fresh sync and refresh storefront cache.
- If review cards render poorly, simplify custom structure/CSS and test incrementally.
