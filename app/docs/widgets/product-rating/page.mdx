# Product Rating

Displays average rating and review count for the current product.

## When to use

Use this widget near the product title, price, or call-to-action so shoppers can quickly assess trust.

## What you can customize

- Visual style
- Rating/count text
- Behavior when there are no reviews
- Optional click behavior to jump to your reviews section

## Typical usage

Place it above the full reviews list and link the interaction so clicking the summary scrolls to detailed reviews.

## Troubleshooting

- If no rating appears, confirm the product has review data.
- If click-to-scroll does not work, re-check the target selector/anchor on the reviews block.
- If styling is inconsistent, verify theme and widget CSS are not conflicting.
