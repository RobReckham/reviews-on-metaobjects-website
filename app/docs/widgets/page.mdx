# Widgets

Widgets control how reviews appear across your storefront.

## Placement from the app

In Shopify admin, open **Widgets** in the app and use **Open in Theme Editor** for the widget you want to place.

## Recommended setup order

1. Add **Product Rating** and **Product Reviews** on product pages.
2. Add **Review Form** where customers can submit feedback.
3. Enable **Product Card Ratings** if you want rating hints on listing pages.
4. Enable **Review Form Page** if you collect reviews via dedicated links.

## Troubleshooting

- If a widget does not render, confirm it is enabled and saved in Theme Editor.
- If layout looks wrong, verify placement in the correct template and section.
- For card and request-page widgets, check their dedicated docs for required integration steps.
