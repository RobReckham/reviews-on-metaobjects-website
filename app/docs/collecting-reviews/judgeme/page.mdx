# Judge.me

Judge.me is treated as an automatic sync source.

## Behavior

- Reviews are discovered and tracked through the app's sync flow.
- Manual import is typically not required.
- New reviews appear in your dashboard and storefront app blocks after sync.

## Important

If reviews are missing, verify the integration is active and recent sync activity completed successfully.

## Troubleshooting

- If no new reviews arrive, check recent sync/webhook activity in the app.
- If only some reviews appear, compare source records with imported product links.
- If updates lag, wait for sync completion before rechecking storefront output.
