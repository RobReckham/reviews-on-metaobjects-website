# Loox

Loox is treated as an automatic sync source.

## Behavior

- Reviews are pulled into the app through the sync flow.
- Manual import is typically not required.
- Synced reviews become available in dashboard and storefront app blocks.

## Important

If review data looks incomplete, confirm the source is connected and sync has finished.

## Troubleshooting

- If reviews stop updating, verify source connection status and recent sync runs.
- If totals differ from source, confirm product mapping and filtering assumptions.
- If storefront stays stale, refresh cache and confirm app blocks read latest imported data.
