# App blocks

App blocks control how reviews appear across your storefront.

## Placement from the app

In Shopify admin, open **App blocks** in the app and use **Open in Theme Editor** for the app block you want to place.

## App embed (FiveOh Core)

Before placing any section blocks, enable the **FiveOh Core** app embed in the Theme Editor. It is required for the review form submission flow and enables a setting to prevent post-purchase review request pages from being indexed by search engines (`Stop review requests from being indexed` — on by default).

Enable it under **Theme Editor → App embeds**.

## Recommended setup order

1. Enable the **FiveOh Core** app embed.
2. Add **Product Rating** and **Product Reviews** on product pages.
3. Add **Review Form** where customers can submit feedback.
4. Enable **Product Card Ratings** if you want rating hints on listing pages.
5. Enable **Review Form Page** if you collect reviews via dedicated links.

## Troubleshooting

- If an app block does not render, confirm it is enabled and saved in Theme Editor.
- If layout looks wrong, verify placement in the correct template and theme section.
- For card and request-page app blocks, check their dedicated docs for required integration steps.
